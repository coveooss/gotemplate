Mathematic Trigonometry

acos

func acos(x interface{}) interface{}

Returns the arccosine, in radians, of x.
Special case is:
    acos(x) = NaN if x &lt; -1 or x > 1

Aliases

  • arcCosine
  • arcCosinus

Examples

Razor:    @ceil(acos(0.5) / 3.1416 * 180)
Template: {{ ceil (mul (div (acos 0.5) 3.1416) 180) }}
Result:   60

atan2

func atan2(x interface{}, y interface{}) interface{}

Returns the arc tangent of y/x, using the signs of the two to
determine the quadrant of the return value.
Special cases are (in order):
    atan2(y, NaN) = NaN
    atan2(NaN, x) = NaN
    atan2(+0, x>=0) = +0
    atan2(-0, x>=0) = -0
    atan2(+0, x&lt;=-0) = +Pi
    atan2(-0, x&lt;=-0) = -Pi
    atan2(y>0, 0) = +Pi/2
    atan2(y&lt;0, 0) = -Pi/2
    atan2(+Inf, +Inf) = +Pi/4
    atan2(-Inf, +Inf) = -Pi/4
    atan2(+Inf, -Inf) = 3Pi/4
    atan2(-Inf, -Inf) = -3Pi/4
    atan2(y, +Inf) = 0
    atan2(y>0, -Inf) = +Pi
    atan2(y&lt;0, -Inf) = -Pi
    atan2(+Inf, x) = +Pi/2
    atan2(-Inf, x) = -Pi/2

Aliases

  • arcTangent2

log1p

func log1p(x interface{}) interface{}

Returns the natural logarithm of 1 plus its argument x. It is more
accurate than log(1 + x) when x is near zero.
Special cases are:
    log1p(+Inf) = +Inf
    log1p(±0) = ±0
    log1p(-1) = -Inf
    log1p(x &lt; -1) = NaN
    log1p(NaN) = NaN

func yn(n interface{}, x interface{}) interface{}

Returns the order-n Bessel function of the second kind.
Special cases are:
    yn(n, +Inf) = 0
    yn(n ≥ 0, 0) = -Inf

    yn(n, x &lt; 0) = NaN
    yn(n, NaN) = NaN

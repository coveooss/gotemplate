acos

func acos(x interface{}) interface{}, error

Aliases: arcCosine, arcCosinus

Returns the arccosine, in radians, of x.
Special case is:
    acos(x) = NaN if x < -1 or x > 1

Example:
[0m[35m    Razor:    [0m @ceil(acos(0.5) / 3.1416 * 180)
[35m    Template: [0m {{ ceil (mul (div (acos 0.5) 3.1416) 180) }}
[35m    Result:   [0m 60

atan2

func atan2(x interface{}, y interface{}) interface{}, error

Aliases: arcTangent2

Returns the arc tangent of y/x, using the signs of the two to determine the quadrant of the return value.
Special cases are (in order):
    atan2(y, NaN) = NaN
    atan2(NaN, x) = NaN
    atan2(+0, x>=0) = +0
    atan2(-0, x>=0) = -0
    atan2(+0, x<=-0) = +Pi
    atan2(-0, x<=-0) = -Pi
    atan2(y>0, 0) = +Pi/2
    atan2(y<0, 0) = -Pi/2
    atan2(+Inf, +Inf) = +Pi/4
    atan2(-Inf, +Inf) = -Pi/4
    atan2(+Inf, -Inf) = 3Pi/4
    atan2(-Inf, -Inf) = -3Pi/4
    atan2(y, +Inf) = 0
    atan2(y>0, -Inf) = +Pi
    atan2(y<0, -Inf) = -Pi
    atan2(+Inf, x) = +Pi/2
    atan2(-Inf, x) = -Pi/2
